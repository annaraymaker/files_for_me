LAPTOP START:
sudo sysctl -w net.ipv4.ip_forward=1
sudo iptables -t nat -A POSTROUTING -o WIFIDEV -j MASQUERADE
sudo iptables -A FORWARD -i enx00XXXX -o WIFIDEV -j ACCEPT
sudo iptables -A FORWARD -i WIFIDEV -o enx00XXXX -m state --state RELATED,ESTABLISHED -j ACCEPT


PI START:
Then on the Pi (over SSH, still sealed), point it at the laptop as gateway and DNS:
sudo nmcli connection modify "Wired connection 1" ipv4.gateway 192.168.50.1 ipv4.dns 8.8.8.8
sudo nmcli connection up "Wired connection 1"
Test from the Pi:
ping -c2 8.8.8.8        # tests routing/NAT
ping -c2 archive.ubuntu.com   # tests DNS
